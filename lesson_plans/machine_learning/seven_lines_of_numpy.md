
# Seven Lines of Numpy

## Goal

Start the lesson by typing 7-10 short phrases of numpy code. 

## Ideas

Each line describes a set of basic 

- create arrays
- create random data
- create row and column matrices
- multiply vectors
- calculate linear, polynomial and exponential functions
- draw random pixels
- draw a flag
- draw a cirle

## Material

You find enough numpy exercises to run 3 courses on: https://burakkagann.github.io/Pixels2GenAI/index.html

## Comments

The way this activity works is that you write the code from your memory.
Seeing you write some non-trivial code fluently is a good way to build authority.

You need to rehearse this at least 2-3 times. On stage, coding skills are worse than at home.
If you haven't done this before, keep notes nearby (but so that students do not see them).

Nevertheless, write slowly. If things go well, they will want to type along.

Comment on what you are writing. Or write comments and play background music.

## Hint

You can do the same with short pieces of basic Python or pandas/polars/plotting.
