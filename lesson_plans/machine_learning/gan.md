
# Generative Models

## Lesson Goal:

Students train one or more generative models.

## Time Frame:

90' or more

## Key Concepts:

- Generative Adversarial Model
- Autoencoder
- Diffusion Model
- U-Net

## Content Delivery:

Use the exercises in Module 12 on https://burakkagann.github.io/Pixels2GenAI/index.html

Choose a few. Each of the chapters take an hour at least.
You should familiarize with the theory and prepare one image that summarizes the main idea.
