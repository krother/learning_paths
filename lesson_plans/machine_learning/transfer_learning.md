
# Transfer Learning

This is a "nice to have" lesson. The topic is somewhat outdated, but the core idea is still relevant: "we can retrain an existing model to do new stuff".

- discuss the mechanics on board using a simplified example (features coming out of the CNN after flattening).
- use ~5 feature names as an example, for understandability label them "elephant ears", "cat eyes", "car wheels" and so on
- label a few images scoring these features 1-5
- now label a new image whose category was not in the original training data

For the practical part, collect some data on a camera, train and predict
